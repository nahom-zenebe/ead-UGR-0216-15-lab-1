A simple **.NET API project** for managing Todos with full CRUD operations and **Swagger documentation**.

## How to Build and Run

1. **Clone the repository:**

```bash
git clone https://github.com/nahom-zenebe/ead-UGR-0216-15-lab-1.git
cd ead-UGR-0216-15-lab-1/TodoApi
```
2. **Build the project:**
   
```bash
dotnet build
```

3. **Run the project:**

```bash
dotnet run
```
4. **Access Swagger UI to explore the API endpoints:**

   ```bash
   http://localhost:5037/swagger

   ```
